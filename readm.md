# Flight Project

## Description
A flight management system with booking and scheduling capabilities.

## Installation
```bash
git clone <repository-url>
cd flight
npm install
```

## Usage
```bash
npm start
```

## Features
- Flight search and booking
- Real-time scheduling
- Passenger management
- Payment processing

## Requirements
- Node.js 14+
- npm 6+
